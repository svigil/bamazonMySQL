# Customer Walk Through

1. Customer will view the current inventory of Bamazon.
2. The customer will then decide how many they'd like to purchase.
3. The total cost will display.
4. The customer will be prompted to either continue shopping or move to checkout.

# Manager Walk Through

1. Manager will choose what they would like to do.
2. They can view the current inventory.
3. They can view the low inventory. (set at less than 250)
4. They can purchase more inventory.
5. They must select which product to re-stock and how many they'd like to re-stock.
6. They can can add a new product to the inventory.
7. They must add in what the item is, what department it will be in, the price, and how many they'd like to add to the inventory.
8. When all tasks are done they can select complete.
